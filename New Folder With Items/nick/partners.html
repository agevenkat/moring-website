Partners

AWS native. Model neutral.

Moring runs on AWS: Bedrock and SageMaker for inference, EKS for runtime, the AWS data stack for retrieval. On the model side we use Anthropic, OpenAI, and open-weight families like Llama and Mistral. We pick the right model for each step in a workflow, and we are not exclusive to any foundation model provider.

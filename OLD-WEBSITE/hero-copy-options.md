# Moring.ai · Hero Copy Options

Four distinct hero variants per solution page. Each takes a different angle so you can pick on positioning, not just wording. Tone and proof points are pulled from `moring-ai-messaging-guide.md` and the live `aidlc.html` / `aiops.html`.

Format for every variant: **Eyebrow · H1 · Sub · Primary CTA · Secondary CTA**.

## How to pick

- **Homepage / brand top-of-funnel:** Variant D on each — strongest voice, least about features.
- **Outbound landing / paid ads:** Variant C on each — sharp pain, fast read.
- **Enterprise sales follow-ups / RFI landers:** Variant B on each — buying-committee or stakes-first framing.
- **Current production:** Variant A on each — already battle-tested in copy.

Each variant keeps the same primary CTA pattern (workshop request) so analytics stay clean across an A/B test.
